## 🧪 Tətbiq Edilən Metodologiyalar

- **Black Box Testing** — Funksionallıq UI vasitəsilə yoxlanılır
- **Positive & Negative Testing** — Düzgün və səhv daxiletmələr üçün davranış yoxlanılır
- **Boundary Value Analysis (BVA)** — Limit dəyərləri test edilir (məs: max say: 9999)
- **Error Guessing** — Form validasiyası və sistem gecikmələri hədəflənir
- **UI/UX Verification** — Mobil, tablet və desktop görünüşü yoxlanılır
- **Security Testing (Edge)** — SQL injection cəhdləri sınaqdan keçirilir
- **Localization Testing** — Azərbaycan dili və UI uyğunluğu yoxlanılır
